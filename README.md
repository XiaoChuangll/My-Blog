### 地址填写示例
填写应用内设置时，需按以下格式拆分地址：
例如：
`http(s)://1.2.3.4:1000`对应
- 主机：1.2.3.4
- 端口：1000
- 协议：http(s)

> 无端口域名的填写规则：
> - http://example.com → 端口填 80，协议填 http
> - https://example.com → 端口填 443，协议填 https
